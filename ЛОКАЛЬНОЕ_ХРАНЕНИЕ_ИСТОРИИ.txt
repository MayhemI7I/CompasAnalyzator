build\bin\

    └── analysis_history.json     ← Файл истории

✅ ПРЕИМУЩЕСТВА ЛОКАЛЬНОГО ХРАНЕНИЯ:

1. ✅ Всё в одном месте
   Программа + История в одной папке

2. ✅ Простой перенос
   Скопировал папку → всё перенесено

3. ✅ Простое резервное копирование
   Одна папка для бэкапа

4. ✅ Нет разрозненности
   Нет поиска по системным папкам

5. ✅ Портативность
   Можно запускать с флешки со своей историей

═══════════════════════════════════════════════════════════════

🔄 КАК ПЕРЕНЕСТИ СТАРУЮ ИСТОРИЮ:

ЕСЛИ у вас была история в старом месте:
C:\Users\<User>\CompasAnalyzer\history\analysis_history.json

СДЕЛАЙТЕ ТАК:

1. Скопируйте старый файл истории:
   C:\Users\МТЦ-01\CompasAnalyzer\history\analysis_history.json

2. Вставьте в новое место:
   C:\Projects\CompasAnalyzer\build\bin\history\analysis_history.json

3. Готово! ✅

═══════════════════════════════════════════════════════════════

📦 ПЕРЕНОС НА ДРУГОЙ ПК (ТЕПЕРЬ ПРОЩЕ!):

БЫЛО (старый способ):
1. Скопировать EXE
2. Найти историю в C:\Users\...
3. Скопировать историю отдельно
4. На новом ПК создать папки
5. Скопировать файлы по отдельности

СТАЛО (новый способ):
1. Скопировать ВСЮПАПКУ build\bin\ на флешку
2. На новом ПК вставить папку
3. Запустить EXE
4. Готово! ✅

═══════════════════════════════════════════════════════════════

💾 РЕЗЕРВНОЕ КОПИРОВАНИЕ (СУПЕР ПРОСТО!):

Скопируйте ОДНУ папку:
build\bin\

Всё! История внутри! 🎉

═══════════════════════════════════════════════════════════════

🚀 ЗАПУСК С ФЛЕШКИ:

Теперь можно:
1. Скопировать папку build\bin\ на флешку
2. Запустить с флешки
3. История будет на флешке
4. Взять флешку на другой ПК
5. Вся история с собой! 📁

🎯 ДЛЯ МИГРАЦИИ СО СТАРОЙ ВЕРСИИ:

Команда для копирования (PowerShell):

Copy-Item "C:\Users\$env:USERNAME\CompasAnalyzer\history\analysis_history.json" `
          -Destination "C:\Projects\CompasAnalyzer\build\bin\history\" `
          -Force
