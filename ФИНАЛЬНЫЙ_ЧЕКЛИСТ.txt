📋 ЧТО ПРОВЕРИТЬ:

═══════════════════════════════════════════════════════════════
1. ДИАЛОГИ ВЫБОРА ДИРЕКТОРИЙ
═══════════════════════════════════════════════════════════════

□ Точечный анализ:
  - Нажать кнопку 📁
  - Должен открыться Windows Explorer
  - Выбрать папку с компасом
  - Путь должен подставиться в поле

□ Пакетный анализ:
  - Нажать кнопку 📁
  - Должен открыться Windows Explorer
  - Выбрать директорию с компасами
  - Путь должен подставиться в поле

□ Редактор файлов:
  - Нажать кнопку 📁
  - Должен открыться Windows Explorer
  - Выбрать директорию
  - Путь должен подставиться в поле

═══════════════════════════════════════════════════════════════
2. ГРАФИК - УПРАВЛЕНИЕ ДИАПАЗОНОМ
═══════════════════════════════════════════════════════════════

□ Выполнить анализ компаса

□ Над графиком должны быть поля:
  - От индекса: [  ]
  - До индекса: [  ]
  - Кнопки: Применить, Сбросить

□ Ввести диапазон (например, 100-300)

□ Нажать "Применить"

□ График должен показать только этот участок

□ Нажать "Сбросить" - должны показаться все данные

═══════════════════════════════════════════════════════════════
3. ГРАФИК - ПОДПИСИ ПРИ НАВЕДЕНИИ
═══════════════════════════════════════════════════════════════

□ Навести курсор на точку графика

□ Должна появиться подсказка:
  ┌────────────────┐
  │ Угол: 45.32°   │  ← ПЕРВЫМ
  │ Индекс: 123    │  ← ВТОРЫМ
  └────────────────┘

═══════════════════════════════════════════════════════════════
4. ГРАФИК - ЦВЕТНАЯ ПОДСВЕТКА ПОВОРОТОВ
═══════════════════════════════════════════════════════════════

□ На графике должны быть видны 4 поворота

□ Каждый поворот своим цветом:
  🔴 Поворот 1 (красный)
  🟢 Поворот 2 (зеленый)
  🟡 Поворот 3 (желтый)
  🟣 Поворот 4 (фиолетовый)

□ В легенде графика должны быть подписи

═══════════════════════════════════════════════════════════════
5. ИСТОРИЯ - ФИЛЬТРЫ
═══════════════════════════════════════════════════════════════

□ Перейти на страницу "История"

□ Над таблицей должна быть панель фильтров:
  - Статус: [Все ▼]
  - От даты: [____]
  - До даты: [____]
  - Сортировка: [Дата (новые первые) ▼]
  - [Применить] [Сбросить]

□ Фильтр по статусу:
  - Выбрать "✓ Успешно"
  - Нажать "Применить"
  - Должны показаться только успешные

□ Фильтр по датам:
  - Выбрать диапазон дат
  - Нажать "Применить"
  - Должны показаться только за этот период

□ Сортировка:
  - Выбрать "Название (А-Я)"
  - Нажать "Применить"
  - Записи должны отсортироваться по алфавиту

□ Сброс:
  - Нажать "Сбросить"
  - Все фильтры должны очиститься
  - Показаться все записи

═══════════════════════════════════════════════════════════════
6. ИСТОРИЯ - ОПТИМИЗАЦИЯ ПАМЯТИ
═══════════════════════════════════════════════════════════════

□ Открыть Диспетчер задач (Ctrl+Shift+Esc)

□ Найти CompassAnalyzer.exe

□ Перейти на страницу "История" (с 3000+ записями)

□ Проверить потребление памяти:
  ✅ Должно быть ~50-100 МБ (НЕ 4 ГБ!)

□ Проверить отсутствие лагов:
  ✅ Прокрутка плавная
  ✅ Фильтры применяются быстро

□ Просмотр записи:
  - Нажать кнопку 👁 (Просмотреть)
  - Должен появиться loading индикатор
  - Через 0.5-1 сек данные загрузятся
  - Откроется страница анализа с графиком

═══════════════════════════════════════════════════════════════
7. ПРОИЗВОДИТЕЛЬНОСТЬ - ПАКЕТНЫЙ АНАЛИЗ
═══════════════════════════════════════════════════════════════

□ Запустить пакетный анализ на 3000 файлов

□ Засечь время обработки

□ Ожидаемый результат:
  ✅ 4-8 минут (было 15 минут)
  ✅ Использование всех ядер CPU
  ✅ Память ~1200 МБ

□ Все результаты должны сохраниться в историю

═══════════════════════════════════════════════════════════════
8. РЕДАКТОР ФАЙЛОВ
═══════════════════════════════════════════════════════════════

□ Перейти на страницу "Редактор файлов"

□ Нажать кнопку 📁 - должен открыться диалог

□ Выбрать директорию

□ Указать текст для удаления (например, "tim")

□ Нажать "Предпросмотр"
  - Должна показаться таблица с превью

□ Нажать "Выполнить переименование"
  - Должно появиться подтверждение
  - После подтверждения - таблица результатов

❓ ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ:

1. Диалоги не открываются:
   → Проверьте консоль (F12)
   → Должно быть: window.go.desktop.App.SelectDirectory

2. Фильтры не применяются:
   → Нажмите "Обновить" на странице истории
   → Проверьте, что есть данные для фильтрации

3. График не перестраивается:
   → Проверьте корректность индексов (0 до max)
   → Начальный индекс < Конечного

4. История потребляет много памяти:
   → Проверьте версию (должна быть PRODUCTION READY)
   → Должен использоваться LoadHistoryMetadata
