ИЛИ проще:

Откройте в текстовом редакторе (Notepad):
   desktop\storage.go

И вставьте код из файла storage.go (см. приложение ниже)

═══════════════════════════════════════════════════════════
ШАГ 2: Обновить desktop/app.go
═══════════════════════════════════════════════════════════

Добавьте в конец файла desktop\app.go:

// GetHistoryDir возвращает путь к папке с историей
func (a *App) GetHistoryDir() (string, error) {
	return GetHistoryPath()
}

═══════════════════════════════════════════════════════════
ШАГ 3: Пересборка
═══════════════════════════════════════════════════════════

set GOMODCACHE=C:\go_modules
set GOCACHE=C:\go_modules\cache
if exist main.go move /Y main.go old_entry_points\main.go.bak
if exist main_tui.go move /Y main_tui.go old_entry_points\main_tui.go.bak
wails build -f main_desktop.go
if exist old_entry_points\main.go.bak move /Y old_entry_points\main.go.bak main.go
if exist old_entry_points\main_tui.go.bak move /Y old_entry_points\main_tui.go.bak main_tui.go

═══════════════════════════════════════════════════════════
ШАГ 4: Запуск
═══════════════════════════════════════════════════════════

build\bin\CompassAnalyzer.exe
